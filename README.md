# FishThread OS

FishThread OS is an offline-first fish trading operating system designed for real-world fish and shrimp businesses.

## Core Principles

- Fully Offline PWA
- Invoice-based workflow
- Purchase → Sale → Ledger flow
- Customer due and advance tracking
- Inventory tracking
- Backup & Restore
- Recycle Bin
- Optimized for iPhone 8
- Bengali-first user experience

---

## Target Platform

- Progressive Web App (PWA)
- iPhone 8 Optimized
- Mobile First
- Offline First

---

## Core Modules

### Dashboard

Business overview and daily summary.

### Purchase

Create and manage purchase invoices.

### Sale

Create sale invoices from purchased inventory.

### Invoices

View purchase and sale invoices.

### Customers

Customer profiles and balances.

### Ledger

Due, payment and advance tracking.

### Collection

Record customer payments.

### Reports

Profit/loss and business reports.

### Backup

Export and restore local data.

### Settings

Theme, language and system preferences.
